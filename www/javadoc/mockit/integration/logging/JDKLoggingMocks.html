mockit.integration.logging

Class JDKLoggingMocks

  • java.lang.Object
    • mockit.MockUp<Logger>
      • mockit.integration.logging.JDKLoggingMocks

  • public final class JDKLoggingMocks
extends MockUp<Logger>
    A mock class containing mocks and stubs for the standard logging API in the JDK (java.util.logging).

    When a test class is annotated as @UsingMocksAndStubs(JDKLoggingMocks.class), all production code touched by the tests in that class will receive mock Logger instances instead of real ones, when one of the factory methods in class java.util.logging.Logger is called.
