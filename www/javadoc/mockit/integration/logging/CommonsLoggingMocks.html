Class CommonsLoggingMocks

  • public final class CommonsLoggingMocks
extends MockUp<org.apache.commons.logging.LogFactory>
    A mock class containing mocks and stubs for the Apache Commons Logging API.

    When a test class is annotated as @UsingMocksAndStubs(CommonsLoggingMocks.class), all production code touched by the tests in that class will receive mock Log instances instead of real ones, when one of the factory methods in class org.apache.commons.logging.LogFactory is called.
